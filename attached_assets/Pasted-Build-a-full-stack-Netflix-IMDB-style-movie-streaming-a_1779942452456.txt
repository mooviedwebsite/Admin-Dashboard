## THEME & UI DESIGN (CRITICAL - Follow Exactly)

### Color Palette:
- Background: #000000 (pure black)
- Surface cards: rgba(255,255,255,0.04) with backdrop-filter: blur(20px)
- Primary accent: #00FF7F (spring green / neon green)
- Secondary accent: #00C853 (darker green)
- Glow effects: 0 0 20px rgba(0,255,127,0.3), 0 0 60px rgba(0,255,127,0.1)
- Text primary: #FFFFFF
- Text secondary: #AAAAAA
- Borders: rgba(0,255,127,0.15)
- Glass cards: border: 1px solid rgba(0,255,127,0.12), background: rgba(255,255,255,0.04), backdrop-filter: blur(24px) saturate(180%)
- NO RED anywhere in the UI

### Design Style:
- Full glassmorphism UI (frosted glass cards, panels, modals, navbar)
- Neon green glow on hover states, active buttons, focused inputs
- Dark grain/noise texture overlay on background (CSS or SVG filter)
- Smooth animations: fade-in, slide-up, scale on hover
- Green glowing borders on focused form inputs
- Glowing green underline on active nav links
- Custom scrollbar: black track, green thumb
- Loading skeleton screens with green shimmer animation
- Movie cards: glass card with green glow on hover, scale(1.05) transform
- Hero section: full-width cinematic banner with gradient overlay (black bottom fade)
- Buttons: solid green (#00FF7F) with black text OR outlined green with glow
- Modals: glassmorphism popup with blurred backdrop
- Font: 'Outfit' or 'Syne' for headings, 'DM Sans' for body (Google Fonts)

### Responsive Design (CRITICAL):
- Mobile (320px–767px): Single column, bottom nav bar, hamburger menu, touch-friendly tap targets minimum 44px, stacked cards
- Tablet (768px–1023px): 2–3 column grid, side nav collapses to icon bar
- Desktop (1024px+): Full sidebar or top nav, 4–5 column movie grid, full hero banner
- Use CSS Grid + Flexbox throughout
- Use clamp() for fluid typography
- All glass cards, modals, and panels must be fully responsive

## CLOUDFLARE D1 DATABASE SCHEMA

```sql
-- Users
CREATE TABLE users (
  id TEXT PRIMARY KEY,
  name TEXT NOT NULL,
  email TEXT UNIQUE NOT NULL,
  password_hash TEXT,
  github_id TEXT,
  avatar TEXT,
  role TEXT DEFAULT 'user',
  is_banned INTEGER DEFAULT 0,
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  last_login DATETIME
);

-- Movies
CREATE TABLE movies (
  id TEXT PRIMARY KEY,
  title TEXT NOT NULL,
  tagline TEXT,
  description TEXT,
  genre TEXT,
  year INTEGER,
  duration INTEGER,
  language TEXT,
  country TEXT,
  director TEXT,
  cast TEXT,
  imdb_rating REAL,
  poster_url TEXT,
  backdrop_url TEXT,
  trailer_url TEXT,
  video_480p TEXT,
  video_720p TEXT,
  video_1080p TEXT,
  video_4k TEXT,
  subtitles TEXT,
  tags TEXT,
  is_featured INTEGER DEFAULT 0,
  status TEXT DEFAULT 'draft',
  views INTEGER DEFAULT 0,
  downloads INTEGER DEFAULT 0,
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- Watch History
CREATE TABLE watch_history (
  id TEXT PRIMARY KEY,
  user_id TEXT,
  movie_id TEXT,
  progress INTEGER DEFAULT 0,
  watched_at DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- Watchlist
CREATE TABLE watchlist (
  id TEXT PRIMARY KEY,
  user_id TEXT,
  movie_id TEXT,
  added_at DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- Reviews
CREATE TABLE reviews (
  id TEXT PRIMARY KEY,
  user_id TEXT,
  movie_id TEXT,
  rating INTEGER,
  comment TEXT,
  status TEXT DEFAULT 'approved',
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- Downloads
CREATE TABLE downloads (
  id TEXT PRIMARY KEY,
  user_id TEXT,
  movie_id TEXT,
  quality TEXT,
  downloaded_at DATETIME DEFAULT CURRENT_TIMESTAMP
);
```
